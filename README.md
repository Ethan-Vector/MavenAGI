# MavenAGI

MavenAGI è un piccolo progetto Java/Maven pensato come playground per logiche di "AGI" (Artificial General Intelligence) super semplificate.

## Requisiti

- Java 17+
- Maven 3+

## Come eseguirlo

```bash
mvn clean package
java -cp target/MavenAGI-0.0.1-SNAPSHOT.jar com.mavenagi.App
```

## Struttura del progetto

- `src/main/java` – codice sorgente principale
- `src/test/java` – test JUnit
- `pom.xml` – configurazione Maven
